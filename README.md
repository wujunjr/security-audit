# ShipAudit

Pre-launch security checks for AI-built applications.

This repository follows the product and implementation baseline in
[`docs/website-security-checkup-master-spec.md`](docs/website-security-checkup-master-spec.md).

## Commands

```bash
pnpm install
pnpm dev
pnpm dev:worker
pnpm typecheck
pnpm test
pnpm build
```

The current implementation covers commercial validation (phase -1) and the
engineering skeleton (phase 0). It does not perform security scans yet.
